LLR-TimeMachine Demo

LLR-TimeMachine is a spectral chronology visualization tool for LLR outputs.

It is not yet included in the current LLR release, but will be soon, so keep checking back. It will prompt you to define a series of coordinates from your LLR outputs to explore visually. It will automatically produce an html file that will allow to do everything featured in this demo for the selected points.

The interface includes:

  • A spectral time series for the point in the "Spectral Trajectory" window
  • A time lapse series of image chips for the 127 pixels bounding each side of the selected point in the "Time Lapse" window
  • The location of the point and bounding rectangle in context according to layers in Google Maps ("Location" window)
  • A time series of image chips in the "Image Chip Gallery" window

The image chips are displayed as uniformly scaled tasseled cap brightness, greeness, wetness (RGB). Areas that are colored funny are either bad data or areas where annual compositing did not result in filling all cloud and shadow holes, resulting in empty data (shown as blue)

The points in the "Spectral Trajectory" window can be displayed by any of the tasseled cap indices that LLR outputs, and the RGB color display can also be changed (use hover-drop buttons). The plot will zoom in/out on the Y axis with a mouse scroll, and will pan with a left-click-hold-drag.

The "Image Chip Gallery" window can be expanded/contracted by left clicking anywhere in it, and the image chips can be zoomed by holding the shift key and scrolling the mouse, or using the zoom slider in the top bar.

Vertices and segments can be added/removed from the spectral trajectory plot by holding the control key and left clicking on a point or an image chip in the "Image Chip Gallery" window. In a later version information about each vertex and segment will be displayed (stay tuned!).
